# 关于博客搭建的一些记录

##### 使用jekyll写博客

- 使用jekyll写博客，我们只需要在项目的**_posts**目录中添加形如2017-07-09-如何使用jekyll.md这样的markdown的文件即可，然后push到github上面。




- 安装jekyll

  > sudo gem install jekyll -v 2.5

- 创建一个新的网站
  > jekyll new my-awesome-site

- 启动打开一个网站
  >  cd my-awesome-site
  >  jekyll serve

- 在_posts文件夹中加入博客
